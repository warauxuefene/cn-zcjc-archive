# cn-zcjc-archive

## 项目简介

本仓库用于归档和发布一系列独立的 HTML 页面。这些页面以静态文件的形式保存，不依赖于特定域名或服务器环境，便于长期查阅与分发。

该仓库不针对任何具体网站或服务，仅作为通用型 HTML 页面存档使用。所有页面内容均以原样存储，不做二次修改或重定向。

## 目录说明

```
.
├── index.html          # 仓库首页（可选）
├── pages/              # 归档页面存放目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 页面相关资源文件（样式、脚本、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：存放所有归档的 HTML 页面文件。
- `assets/`：存放页面所需的辅助资源，如样式表、脚本和图片等。

## 页面归档说明

- 所有归档页面均为独立的 HTML 文件，可直接在浏览器中打开查看。
- 页面内容保持原始状态，不进行内容修改或功能增强。
- 归档文件可能包含外部资源引用（如字体、图片等），部分资源可能因网络环境无法正常加载。

## 维护说明

- 本仓库仅用于归档和发布，不提供内容更新或技术支持。
- 如有新增页面需要归档，请提交 Pull Request，将 HTML 文件放入 `pages/` 目录。
- 不接受对已归档页面内容的修改请求。
- 仓库维护者保留对归档内容进行整理或移除的权利。

## 许可

本仓库的归档内容遵循其原始来源的许可协议。仓库本身的组织结构和 README 文档采用 MIT 许可证，详见 [LICENSE](./LICENSE) 文件。
